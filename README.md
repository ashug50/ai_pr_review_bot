# AI PR Review Bot

Simple AI-powered PR review bot for data engineering.

## Features
- Reviews code diffs
- Suggests improvements (Code Optimisation/ PySpark Performance)
- Suggests unit tests
### Next Features 
- Goveranance Rules
- Meta Data Updates and Suggestions

## Setup

```bash
pip install -r requirements.txt
export OPENAI_API_KEY=your_key
```

## Run

```bash
python src/review_bot.py --repo_path <repo>
```
